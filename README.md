## Calcolator Chat Application

#### Whats included?

- webpack-dev-server
- webpack 4
- eslint configuration + prettier 
- redux
- redux-saga
- sass
- jest
- babel
- hot reload


## Quick Use

Run in locally - webpack-dev-server

```
npm start
```

Run tests / test in dev mode (--watch)

```
npm run test
```

Fix eslint issues

```
npm run lint-fix
```
